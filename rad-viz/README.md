# RAD-Sim Visualization

## Quick Start

```bash
cd /path/to/rad-viz
npm install
npm run dev # debug mode
```

## Deployment

```bash
cd /path/to/rad-viz
npm run build
python3 -m http.server -d dist # http://localhost:8000
```
